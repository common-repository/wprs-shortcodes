Display any of the Box markup elements with layout via a simple shortcode. An add-on for WPRichSnippets plugin.

== Description ==

This add-on allow you to use a shortcode to display Box elements by entry id, which can be used on any post, page, widget or in theme files. This is a cool and light add-on for the [WPRichSnippets](https://wprichsnippets.com/) plugin, which will allow you to establish amazing customization on your WPRichSnippets enabled site.

== Installation ==

1. Upload the entire `wprs-shortcodes` folder to the `/wp-content/plugins/` directory
2. DO NOT change the name of the `wprs-shortcodes` folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. You are all set.
